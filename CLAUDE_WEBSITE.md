# Website Instructions (Scoped)

This file governs modifications related only to the portfolio website UI.

Scope is limited to:

- index.html
- Inline styles
- CSS files
- Tailwind classes (if present)
- Profile picture styling
- Layout improvements
- Typography improvements
- Animations (lightweight only)

Do NOT modify:
- Backend logic
- Other folders
- Build tools
- Deployment configs

---

# Objective

The website already exists.

The goal is:
- Improve visual quality
- Improve readability
- Improve spacing and alignment
- Improve color harmony
- Improve responsiveness
- Improve profile picture presentation

WITHOUT rewriting everything.

---

# Operating Rules

1. First understand current structure.
2. Do not rewrite entire file.
3. Keep HTML semantic.
4. Avoid unnecessary div nesting.
5. Improve spacing using consistent scale.
6. Avoid overly dark themes unless requested.
7. Maintain a professional aesthetic.
8. Use smooth transitions only if subtle.

---

# Profile Picture Rules

- Should feel intentional, not random.
- Proper aspect ratio.
- Clean border radius.
- Soft shadow.
- Proper contrast against background.
- Responsive scaling.
- Avoid pixelation.
- Center alignment unless design requires otherwise.

---

# Theme Guidelines

If adjusting theme:
- Use consistent primary color.
- Avoid color chaos.
- Ensure contrast accessibility.
- Keep text readable.
- Avoid mixing too many accent colors.

---

# Layout Improvement Guidelines

- Improve hero section first.
- Align content grid.
- Maintain equal padding on both sides.
- Improve typography hierarchy:
  - H1 dominant
  - H2 clear
  - Body readable
- Avoid cramped spacing.
- Add breathing room.

---

# Animations

If adding:
- Typing effect allowed.
- Fade-in allowed.
- Hover micro-interactions allowed.
- No heavy animation libraries.

Subtle > flashy.

---

# Self-Improvement Loop (Website Version)

When improving:

1. Identify visual issue
2. Improve only that part
3. Check responsiveness
4. Ensure consistency
5. Move to next section

---

# What Good Looks Like

- Clean
- Minimal
- Intentional
- Professional
- Smooth
- Modern but not trendy

---

# Bottom Line

You are not redesigning from scratch.

You are refining.

Make it sharper.
Make it cleaner.
Make it premium.
